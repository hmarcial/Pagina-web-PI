System of Monitoring in Greenhouses

Objective

Crear un nodo para el monitoreo aplicado a la agricultura de precisión, conectado a un sistema de escritorio con despliegue de información a través de un servidor Web.

Requirements

  1. Instalación del Sistema Operativo y los Servicios de Internet.
  2. Programa de graficación y reportes en un lenguaje visual
  3. Modelo matemático del comportamiento de una variable ambiental.
  4. Programación del método numérico más adecuado para el análisis de los datos.
  5. Implementación de servicios en un sistema embebido.

Materiales

Software

  • Sistemas Operativos Windows.
  • Servidores y clientes de FTP y HTTP.
  • Visual Studio 2017.
  • Matlab.
  • Procesador de texto.
  • Para realizar diagramas y presentaciones visuales.

Sistemas Embebidos

  • Tarjeta de desarrollo.
  • Fuente de poder.
  • Cable convertidor USB a RS-232.
  • Sensores de: humedad y temperatura, luz ambiental y humedad del suelo.
  • Aplicaciones de diseño y simulación electrónica.
  • Compilador del lenguaje C.
  • Terminal serial.
